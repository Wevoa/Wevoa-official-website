Documentation

Get from install to running app quickly.

These are the core paths new developers need first: install the runtime, generate a starter, understand the request flow, and ship with the built-in production commands.

01

Get started

Use the installer or runtime binary, then create and run a starter project immediately.

  • wevoa --version
  • wevoa create dashboard app
  • cd app
  • wevoa start

02

Build pages with Wevoa

Each request runs through routes, templates, and structured helpers without requiring frontend JavaScript.

  • route "/" { return view("home.wev", data) }
  • {{ expression }} interpolation
  • extend, section, include, and component support
  • request, session, params, and json() helpers

03

Ship safely

Production support is part of the runtime, not bolted on in another tool.

  • wevoa build
  • wevoa serve
  • wevoa doctor
  • SHA256 release verification
